Guide lines to Publish the post
  1. Enter Title field of the post related to your content.
    Example: Python is programming language.
  2. Enter Slug field same as title but each word separated by '-' symbol.
    Example: Python-is-programming-language.
  3. Select Author from drop down list.
    Login User name must be matched selected Author name
  4. Write some description about the post in the Body field.
  5. If you have data in file chose the file in Data field.
    File types are .doc or .docx or .txt or .ppt
  6. If you have code in file chose the file in Code field.
    File types are .py or .c or .java or .js or e.t.c
